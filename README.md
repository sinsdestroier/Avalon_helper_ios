# Avalon Helper iOS

這是一個用 SwiftUI 製作的《阿瓦隆 / The Resistance: Avalon》iOS 輔助程式，目標是讓玩家可以在同一台裝置上完成建房、發身份、任務流程、投票、刺殺與結果揭露，並加入大量梗圖、音效與開場動畫演出。

## 功能概覽

- 建立房間與設定玩家人數
- 自訂角色組合
- 調整投票顯示策略與規則
- 身份逐位揭露
- 任務提名、投票、出牌、結果公布
- 刺殺階段與結算畫面
- 音效與特效開關
- 開場影片動畫
- 多種任務成功 / 失敗 / 刺殺 / 壞人勝利演出
- Debug 模式快速進入指定流程

## 專案結構

- `阿瓦隆/AvalonApp.swift`
  App 入口。
- `阿瓦隆/ContentView.swift`
  開場動畫與主畫面切換。
- `阿瓦隆/SetupView.swift`
  建房、規則設定、玩家名稱輸入、Debug 模式入口。
- `阿瓦隆/IdentityRevealView.swift`
  身份逐位揭露流程。
- `阿瓦隆/RoundFlowView.swift`
  核心遊戲流程，包含投票、任務、刺殺、結算與各種演出。
- `阿瓦隆/GameSession.swift`
  遊戲狀態與規則推進。
- `阿瓦隆/GameModels.swift`
  角色、陣營、規則與設定模型。
- `阿瓦隆/SoundEffectPlayer.swift`
  音效與背景音樂播放控制。
- `阿瓦隆/AppSettings.swift`
  音效 / 特效開關狀態。
- `阿瓦隆/Assets.xcassets`
  圖片、梗圖、棋盤與其他視覺素材。
- `阿瓦隆/sound/`
  音效與背景音樂素材。
- `阿瓦隆/video/`
  開場影片素材。

## 目前流程

1. App 啟動後播放開場影片。
2. 淡入進入建立房間畫面。
3. 設定玩家與規則後開始遊戲。
4. 逐位揭露身份。
5. 進入任務流程：
   提名 -> 投票 -> 任務出牌 -> 任務公布。
6. 若好人先達成 3 次成功，進入刺殺階段。
7. 若壞人先達成 3 次失敗，直接進入壞人勝利演出與結算。

## 音效與特效

程式內已支援：

- 首頁 / 身份揭露背景音
- 一般按鈕點擊音
- 任務成功 / 失敗演出
- 刺殺成功 / 失敗演出
- 壞人三次任務失敗勝利演出
- 音效與特效獨立開關

素材目前放在：

- `阿瓦隆/sound/`
- `阿瓦隆/video/`
- `阿瓦隆/Assets.xcassets`

## Debug 模式

首頁右上角齒輪選單內提供 `DEBUG模式`，可快速：

- 快速發身份
- 直接進入回合
- 直接進投票
- 載入 5 人預設測試局
- 載入圖示排版預覽

## 開發需求

- Xcode
- iOS SwiftUI 專案環境
- 建議使用最新版 Xcode 開啟與建置

## 執行方式

1. 用 Xcode 開啟專案。
2. 確認 `sound/`、`video/` 與 `Assets.xcassets` 內素材都有加入 app target。
3. 選擇模擬器或實機。
4. Build / Run。

## 備註

- 本專案包含大量自訂圖片、音效與影片素材，若素材檔名變動，需同步修改程式中的資源名稱。
- 部分演出依賴 bundle 內媒體檔案，若素材未加入 target，執行時會出現找不到資源但不一定會直接編譯失敗。
- 若要調整規則、音效映射或特殊演出，優先查看 `RoundFlowView.swift` 與 `SoundEffectPlayer.swift`。
